VueJS utilities, libraries and Vue components in use across @polkadot projects

vue-identicon

A flexible identity icon generator supporting a number of display styles based on theme.

vue-keyring

An extension to the base keyring that allows for additions such as browser storage.

vue-utilities

Utilities for formatting and working with settings inside any browser-based application.

# Welcome

This is a collection of Vue.js libraries, utlities and classes that allows for easier use of the Polkadot JS infrastructure inside your apps.

This repository split up into a number of internal packages, namely -
